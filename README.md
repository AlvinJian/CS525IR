# Run it at first time
```
cd client-app/
npm install
npm run build
```
the above commands are for react. If the front-end code is changed, you have to re-run `npm run build` in `client-app/`
# Run!
make sure `flask` is installed, then do:

on Windows:
```
./run.ps1
```

on Linux/Mac:
```
./run.sh
```
